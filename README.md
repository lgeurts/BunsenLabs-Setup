## Setup

- Application Launcher: `rofi`
- Bar: `lemonbar`
- Compositor: `compton`
- Music Player: `spotify` / `mopidy` + `ncmpcpp`
- Notifications: `dunst`
- Shell: `bash`
- Terminal Emulator: `urxvt`
- Text Editor: `neovim` / `vim`
- Video Player: `mpv`
- Web Browser: `Firefox (Nightly)`
- Window Manager: `i3-gaps` / `openbox`
